Free Hosting Setup Guide
Host this platform free on GitHub Pages in under 10 minutes
Step-by-step: GitHub Pages
1
Go to github.com and create a free account (if you don't have one)
2
Click New Repository. Name it cpr-trader. Set it to Public. Click Create.
3
Click uploading an existing file. Drag and drop the trading-platform.html file Claude gave you.
4
Rename the file to index.html before uploading. Then click Commit changes.
5
Go to Settings → Pages. Under Source, select Deploy from branch → main → / (root). Click Save.
6
Wait 2 minutes. Your site will be live at https://yourusername.github.io/cpr-trader — bookmark it!
Important Notes
✅  This is a completely free website — no monthly fees ever
✅  Trade journal data is saved in your browser's local storage — stays on your device
✅  Live prices use Delta Exchange public API — no login or API key needed
⚠️  If you clear browser data, journal entries will be erased — export regularly
⚠️  Prices are for reference only — always confirm on Delta Exchange before trading
📱  The site works on mobile too — open it on your phone while trading
